# 粒子艺术生成器

一个基于 HTML5 Canvas 的交互式粒子艺术生成器，支持多种预设模式和自定义参数调整。

## 功能特性

### 🎨 预设模式
- **星空模式**：繁星点点的夜空效果，粒子缓慢漂移
- **烟花模式**：绚丽的烟花绽放效果，定时发射
- **水流模式**：流动的水粒子效果，带有拖尾

### 🎛️ 可调节参数
- **粒子数量**：50 - 500 个
- **粒子颜色**：自定义颜色选择器
- **粒子速度**：0.1 - 5.0 倍速
- **粒子大小**：1 - 10 像素
- **发光强度**：0 - 30 级辉光效果

### 🖱️ 鼠标交互
- **跟随模式**：粒子被鼠标吸引
- **散开模式**：粒子被鼠标排斥

### 📸 其他功能
- **截图保存**：一键保存当前画布为 PNG 图片
- **重置参数**：快速恢复默认设置
- **响应式布局**：支持不同屏幕尺寸

## 技术栈

- **HTML5 Canvas**：高性能粒子渲染
- **原生 JavaScript**：粒子系统逻辑
- **CSS3**：渐变、发光效果、动画

## 项目结构


## 使用方法

### 直接运行
直接在浏览器中打开 `index.html` 文件即可使用。

### 使用本地服务器
推荐使用本地服务器运行以获得更好的体验：

```bash
# Python 3
python -m http.server 8080

# 或 Node.js
npx serve .
```

然后在浏览器中访问 `http://localhost:8080`。

## 核心实现

### Particle 类
单个粒子对象，包含：
- 位置、速度、大小、颜色等属性
- 生命周期管理
- 拖尾效果
- 发光渲染

### ParticleSystem 类
粒子系统管理器，负责：
- 粒子的创建与销毁
- 动画帧循环
- 鼠标交互处理
- 模式切换
- 参数更新

### 性能优化
- 使用 `requestAnimationFrame` 实现流畅动画
- Canvas 半透明覆盖实现拖尾效果
- 对象池复用粒子
